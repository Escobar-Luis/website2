What I Learned

I want to dedicate this page to write down useful things I learned while actually coding my first website!

*Quick Note* I use Mac OS so accessing the cool features would probably different in other OS.
  • Multiple Pages
    • We have to create new html files using touch operator. Open it in the browser. Then link them in the nav bar by pasting link into href
  • Wrap Text in VSC
    • Open the file you want to use and click the view tab on top. Then click "Toggle Word Wrap" and viola! No matter how small your code window is, the code won't ever leave the screen
